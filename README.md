# Grassflog


## First-time dev setup

  * Copy `config/secrets.exs.template` to `config/secrets.exs`. Fill in correct env vars as necessary.
  * `mix ecto.setup`
  * `cd assets && yarn && cd ..`
  * `mix phx.server`
  * Go to http://localhost:4000 and confirm the local site works
  * `mix test` to run all tests
